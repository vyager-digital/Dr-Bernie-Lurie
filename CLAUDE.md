# Dr. Bernie Lurie — Project Notes

## Overview
- Naturopathic doctor site: "Nature's Healing Process"
- Two pages, no external CSS file — all styles are embedded in each HTML file
- Status: in progress

## Pages
- `index.html` — main site (general practice)
- `hormonal/index.html` — dedicated page for hormonal harmony / women's health

- Fonts: Cormorant Garamond (body/headings), Josefin Sans (accents)
- Tone: natural, earthy, elegant
- Color palette:

  - --moss: #2e5c3e
  - --sage: #6b8f71

- Tone: soft, feminine, warm
